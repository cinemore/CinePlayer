## ADDED Requirements

### Requirement: iOS and macOS settings panel SHALL use Cinemore play-settings structure
The settings panel on iOS and macOS SHALL use the Cinemore play-settings style structure with grouped rows and clear section hierarchy.

#### Scenario: Settings panel shows grouped play settings
- **WHEN** the user opens settings panel
- **THEN** the panel MUST render grouped play-setting sections rather than a plain linear list
